## npm-指令

```
表示全局安装 npm 包
npm install -g

将其命名为cnpm
npm install -g cnpm

全局环境下安装 Vue CLI 工具
npm install -g @vue/cli

全局安装webpack
npm install webpack -g

```

```
列出项目依赖
npm list 

列出全局依赖 
npm list -g

只列出顶级依赖信息
npm list --depth=0
```

```
查看版本
npm -v

配置阿里云镜像源
npm config set registry http://registry.npm.alibaba-inc.com

配置清华镜像源（better）
npm config set registry https://registry.npmmirror.com/

查看下载源
npm config get registry
```
