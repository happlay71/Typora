# 运行uWSGI

![](https://happlay-docs.oss-cn-beijing.aliyuncs.com/docs/Snipaste_2024-03-16_17-10-01.png)

```
whereis uwsgi
uwsgi --ini uwsgi.ini
uwsgi --stop uwsgi.pid
```

**查看进程**

```
ps aux|grep 'uwsgi'
```
