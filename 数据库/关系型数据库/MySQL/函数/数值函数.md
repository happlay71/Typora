| 函数       | 功能                               |
| ---------- | ---------------------------------- |
| ceil(x)    | 向上取整                           |
| floor(x)   | 向下取整                           |
| mod(x,y)   | 返回x/y的模                        |
| rand()     | 返回0~1内的随机数                  |
| round(x,y) | 求参数x的四舍五入的值，保留y位小数 |

```mysql
# 数值函数
-- ceil
select ceil(1.1);
-- floor
select floor(1.9);
-- mod  前%后
select mod(100, 11);
-- rand
select rand();
-- round  四舍五入
select round(2.345, 2);
# 生成六位随机验证码
select lpad(round(rand() * 1000000, 0), 6, '0');
```
