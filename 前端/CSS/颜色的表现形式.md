| 表示方式       | 表示含义                          | 取值                                       |
| -------------- | --------------------------------- | ------------------------------------------ |
| 关键字         | 预定义的颜色名                    | red、green、blue……                         |
| rgb表示法      | 红绿蓝三原色，每项取值范围：0-255 | rgb(0,0,0)、rgb(255,255,255)、rgb(255,0,0) |
| 十六进制表示法 | #开头，将数字转换成十六进制       | #000000、#ff0000\#cccccc,简写：#000、#ccc  |

```css
color: red;
color: rgb(255, 0, 0)
color: #f00;
```
